About



This is a simple MEAN stack application that demonstrates Two-Factor Authentication with Authy.


Two-Factor Authentication is enabled for both login and registration. The following methods are available.

  • OneTouch - If you have registered with an Authy app, you will be able to approve a login or registration request by one tap
    on the Authy app's OneTouch screen.

  • SoftToken - If you have registered with an Authy app you can choose to use the generated SoftTokens to verify.

  • SMS - If you haven't registered with an Authy app, an SMS will be sent to your mobile number.

Note: A valid country code and phone number are required in the registration process, to continue with logging in.
